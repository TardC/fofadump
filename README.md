# Fofadump

## Installation Instructions

```shell
go get -v github.com/tardc/fofadump/cmd/fofadump
```

## Usage

```
Usage of fofadump:
  -email string
        Email of fofa account
  -fields string
        Fields to query (default "host,ip,port")
  -key string
        Key of fofa account
  -q string
        Fofa query rule
  -size int
        The number size to query (default 100)
```
